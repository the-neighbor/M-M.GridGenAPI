M-M.GridGen

A CSS Grid Layout Generator

This webapp contacts /api/gen with a get request. Query parameters "rows" and "columns" are used to specify the dimensions of the grid, and the query parameter "pieces" is used to specify the number of divs to use to fill the grid. The API then uses a recursive backtracking algorithm to find an arrangement of all the possible rectangular pieces that would fit in the grid in order to fill the grid using exactly the number of pieces specified. The order in which it tries pieces is also shuffled each time, so each time you click submit you should get a different layout, even with the same parameters. Try it out below!
